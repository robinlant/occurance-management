# Duty Robin

## Problem

When some things need to be done and there are big amount of people who do it, its often hard to keep track
who did exactly how many times and how is slacking. Also it is often managed in excel spredsheets which in a
long term can make data hard to read

## Features

- Create an occurance group (for example Versandaktion, Empfangvertretung, etc)
- Create an occurance and put it into a group or without it
- Users can enter their out of office periods
- If occurance still has free spots notify via Mail users
- Occurances can have different number for minimum and maximum participants
- View a leaderboard of who participated the most
- Different groups like orginizers ( who dont have to participate ) and those who have
